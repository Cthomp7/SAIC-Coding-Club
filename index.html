Welcome to the SAIC Coding Club!

This club is for any SAIC students interested in learning and/or talking about code.


All skill levels from beginner to expert are welcome! So if you have never written a line of code, but need a little help getting started we can help!

When?

The SAIC Coding Club is held on alternating Fridays and Thursdays.

Thursdays: 4pm-6pm | Fridays: 3pm-5pm

Where?

These meetings are held in the Flex Space located on the 4th floor (Room 400) in the Maclean building.
